Smart Face Detect

Smart Face Detect app that runs a facial recognition scan, detects a human face on an image and draws a face detecton box around a face it detects. It can detect as many faces as possible on an image but its face box is limited to one face. Users have to signin or register if they don't have an account in order to use the app. A count of how many scans run by each user is recorded.

The following were used to create this webapp:

Front-End: Html, Css, Javascript, ReactJs, Tachyons, Bootstrap

Backend: NodeJs, ExpressJs, Clarifai AI Machine Learning Model

Database: PostgreSQL
